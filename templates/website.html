Task Manager

What is it about?

Task manager is a tool allowing its users the advantage to improve their time managing skills. It gives users the ability to manage their tasks and assignments to better keep track of them and their time. It offers a variety of descriptors for assignments and allows users to set a date and time for which a task should be complete, which the user will be reminded of an hour or a few hours beforehand based on their preference.

Overall, it is a neat project that helps anyone who is always in need of knowing what assignment is due closer to the due date than other assignments.

Purpose?

The Purpose of this project was to help anyone who are lacking the time and the self goal setting to do these tasks at an appropriate time.

How will this benefit the user?

This program will allow the user to improve their problem solving skills and help increase their efficiency within meeting the due dates. in conclusion, it helps increase productivity to make ends meet.

Modules Used

Flask, Pygui, Datetime, pandas

These are the modules used to make this project happen. as well as aesthetically making it user-friendly.
